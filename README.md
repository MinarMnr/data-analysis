# E-Survey Frontend

### Instructions:

- Clone this repository:  
  `git clone https://eatlbd.net/banbeis-ieims/data-analysis-frontend.git`
- Go to the folder directory  
  `cd data-analysis-frontend`  
  `git checkout dev`
  `git checkout -b your branch name(shawon)`
- Install dependencies  
  `npm install`
- Run application  
  `npm start`
